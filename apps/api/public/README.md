### This is a simple app that allows you to compute the mod-3 remainder of a binary string using a finite state machine (FSM) implemented in the `@aashack/fsm` package. It provides an API endpoint for computing the remainder and includes end-to-end tests using Playwright.

### There isn't a need to build anything, just run `npm install` to set up dependencies and then `npm run dev` to start the server. The API will be available at http://localhost:3000, and you can send POST requests to `/api/mod3` with a JSON body containing the binary string input.
